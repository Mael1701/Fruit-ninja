# Fruit Ninja Game

Welcome to the Fruit Ninja Game! This is a browser-based game where players need to slice falling fruits and avoid slicing bombs. The game tracks the player's score and provides feedback when the game is over.

## Gameplay

The objective of the game is to slice as many falling fruits as possible while avoiding bombs. The player earns points for each fruit sliced and loses the game if they slice a bomb or miss a fruit.

## Features

- Interactive slicing of fruits and bombs.
- Score tracking and game over feedback.
- Customizable game settings for number of fruits and bombs.

## How to play

1. **Start the Game:**
   - Follow this link https://mael1701.github.io/Fruit-ninja/#.

2. **Slicing Fruits:**
   - Use your mouse to click and drag across the screen to create a slicing motion.
   - When your mouse trail intersects with a fruit, the fruit will be sliced, and you will earn one point.

3. **Avoiding Bombs:**
   - Bombs will also fall from the top of the screen.
   - Avoid slicing bombs as they will end the game immediately.
   - Focus on slicing only the fruits to keep playing and increase your score.

4. **Game Over:**
   - The game will end if you miss a fruit or slice a bomb.
   - When the game is over, a modal will appear displaying your final score and a message.
   - Click the button in the modal to restart the game.

5. **Resetting the Game:**
   - Upon clicking the button, the game will reset.
   - New fruits and bombs will start falling from the top of the screen.
   - Your score will be reset to zero, and you can start slicing fruits again.

6. **Scoring:**
   - Each fruit sliced adds one point to your score.
   - The score is displayed at the left of the screen and updates in real-time as you slice fruits.

Enjoy the game and aim for the highest score possible!
